# logica-programacao-c
Exercícios práticos em linguagem C com foco em lógica de programação e programação estruturada.
Este repositório contém exercícios e atividades acadêmicas desenvolvidas durante os estudos em programação, com foco em lógica de programação e fundamentos da linguagem C.

Os códigos foram criados como parte de atividades de faculdade e exercícios práticos, com o objetivo de reforçar conceitos básicos da programação estruturada em C.

Conteúdos praticados

Declaração e uso de variáveis

Entrada e saída de dados (scanf e printf)

Estruturas condicionais

Estruturas de repetição

Funções sem retorno (void)

Vetores e matrizes

Manipulação de dados em memória

Arquivos do repositório

variaveis.c – Exercícios introdutórios sobre declaração de variáveis, leitura de dados e exibição de resultados.

funcoes_sem_retorno.c – Implementação e uso de funções do tipo void, reforçando modularização do código.

matriz.c – Manipulação de matrizes, percorrendo elementos e realizando operações básicas.

Objetivo

Praticar lógica de programação e desenvolver o raciocínio lógico utilizando a linguagem C, servindo como base para disciplinas acadêmicas e para o aprendizado de programação estruturada.
