# Programação em C

Este repositório contém exercícios e atividades acadêmicas desenvolvidas em linguagem C durante os estudos de programação, com foco em lógica de programação, estruturas de dados básicas e organização de código.

Os arquivos foram criados como parte de atividades práticas da faculdade, com o objetivo de consolidar conceitos fundamentais da linguagem C.

## Conteúdos praticados
- Declaração e uso de variáveis
- Estruturas condicionais
- Estruturas de repetição
- Funções com e sem retorno
- Manipulação de vetores e matrizes
- Organização de código em arquivos
- Introdução a Tipos Abstratos de Dados (TAD)
- Estruturas de dados básicas (listas e filas)

## Objetivo
Praticar lógica de programação, compreender os fundamentos da linguagem C e desenvolver raciocínio lógico aplicado à resolução de problemas computacionais.
